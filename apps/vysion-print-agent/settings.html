Vysion Print Agent

Eenmalig instellen — daarna start de kassa automatisch met de juiste printer.

Status controleren…

1 · Jouw zaak

De code van jouw zaak op Vysion (deel achter /shop/ in de URL). De volledige kassa-URL wordt automatisch opgebouwd.

2 · Bonprinter

USB of netwerk maakt niet uit — kies de printer zoals Windows hem ziet. Niet “Microsoft Print to PDF”: dat is geen echte bonprinter. Bij “Opslaan” wordt je printerdirect bewaard, ook als de tenant hieronder nog niet ingevuld is.
